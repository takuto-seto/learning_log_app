<h1>Learning Log App</h1>
日々の学習を、美しく、シンプルに記録する。 個人の学習進捗を管理するためのフルスタック・Webアプリケーションです。

<h2>Demo<h2>
https://learning-log-app-frontend.onrender.com/

<h2>主な機能<h2>
認証システム: JWTを利用したセキュアなログイン/サインアップ機能

CRUD操作: 学習ログの作成、読み取り、更新、削除

検索・フィルタリング: タイトルやカテゴリーによるリアルタイム検索

ソート機能: 投稿日時による昇順・降順の並び替え

レスポンシブデザイン: PC、タブレット、スマホの全デバイスに最適化されたUI

インタラクティブUI: Intersection Observerを活用したスクロールアニメーション


<h2>技術スタック<h2>
<h3>Frontend<h3>
React 18 / TypeScript

Tailwind CSS (スタイリング)

Vite (ビルドツール)

<h3>Backend<h3>
Python 3.9+ / FastAPI

SQLAlchemy (ORM)

PostgreSQL

JWT Authentication

<h3>Infrastructure / DevOps</h3>
Docker / Docker Compose

Render (Hosting)
